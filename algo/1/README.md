–ö–∞–∂–¥—ã–π –æ—Ç–≤–µ—Ç –∏–Ω—Ç–µ—Ä–∞–∫—Ç–æ—Ä–∞ –∑–∞–≤–µ—Ä—à–∞–µ—Ç—Å—è –ø–µ—Ä–µ–≤–æ–¥–æ–º —Å—Ç—Ä–æ–∫–∏. –ö–∞–∂–¥—ã–π –∑–∞–ø—Ä–æ—Å –≤–∞—à–µ–π –ø—Ä–æ–≥—Ä–∞–º–º—ã –¥–æ–ª–∂–µ–Ω –∑–∞–≤–µ—Ä—à–∞—Ç—å—Å—è –ø–µ—Ä–µ–≤–æ–¥–æ–º —Å—Ç—Ä–æ–∫–∏ –∏ —Å–±—Ä–æ—Å–æ–º –±—É—Ñ—Ñ–µ—Ä–∞ –ø–æ—Ç–æ–∫–∞ –≤—ã–≤–æ–¥–∞ (`cout.flush()` –≤ C++, `sys.stdout.flush()` –≤ Python).
