# [I. Остовное дерево](I.cpp)

| Ограничения                                 |
|:-------------------------------------------:|
| ограничение по времени на тест: 8 секунд    |
| ограничение по памяти на тест: 256 мегабайт |
| входной файл: `стандартный ввод`            |
| выходной файл: `стандартный вывод`          |

## Условие

Даны точки на плоскости, являющиеся вершинами полного графа. Вес ребра равен расстоянию между точками, соответствующими концам этого ребра. Требуется в этом графе найти остовное дерево минимального веса.

## Входные данные

Первая строка входного файла содержит натуральное число $n$ — количество вершин графа $(1 \leqslant n \leqslant 10000)$. Каждая из следующих $n$ строк содержит два целых числа $x_i$, $y_i$ — координаты $i$-й вершины $(-10000 \leqslant x_i, y_i \leqslant 10000)$. Никакие две точки не совпадают.

## Выходные данные

Первая строка выходного файла должна содержать одно вещественное число — вес минимального остовного дерева.

## Примеры

**входные данные**:

```text
2
0 0
1 1
```

**выходные данные**:

```text
1.4142135624
```
